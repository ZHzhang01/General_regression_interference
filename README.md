Here is my code for the work ``General auxiliary variables with approximate neighbourhood interference''.

It contains two parts: synthetic experiments and real-world experiments:

1) Synthetic experiments: (i) syn_bn_3: present the result for bn = 3; (ii) syn_total: present results for all kinds of bn $\in \{0,1,2,3\}$. In our main paper, we focus on $bn = 3$.
We choose the target population as 3000.

2) Real-world experiments: ex_bn_``X''. Here X $\in \{0,1\}$. Each folder contains the result, which is recorded in ``ex_result''.
